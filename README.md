# SOMETIME

## Giới thiệu

**Sometime** là một ứng dụng web đăng bài blog đơn giản, được xây dựng phục vụ cho môn học tại trường. Ứng dụng cho phép người dùng tạo, chỉnh sửa và xem các bài viết blog trên cùng một nền tảng.

## Mục tiêu dự án

* Thực hành phát triển web fullstack
* Làm quen với mô hình client–server
* Áp dụng kiến thức về frontend, backend và cơ sở dữ liệu

## Công nghệ sử dụng

### Frontend

* **Vue.js**: xây dựng giao diện người dùng
* HTML, CSS, JavaScript

### Backend

* **Java Servlet**: xử lý logic nghiệp vụ và cung cấp API

### Cơ sở dữ liệu

* **MySQL**: lưu trữ dữ liệu người dùng và bài viết blog

## Chức năng chính

* Đăng bài blog
* Xem danh sách và nội dung bài viết
* Chỉnh sửa / xóa bài viết
